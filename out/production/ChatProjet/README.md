# Mini Projet Chat Java Socket

## Description
Application de chat client-serveur en Java utilisant les sockets TCP.
Supporte 3 modes de fonctionnement:
- Mode 1: Client ↔ Serveur
- Mode 2: Communication client-client via serveur (serveur passif)
- Mode 3: Communication client-client et client-serveur

## Fonctionnalités
- Chat public (broadcast)
- Messages privés
- Interface graphique Swing
- Liste des utilisateurs connectés
- Historique des conversations
- Fenêtres de chat privé séparées

## Prérequis
- Java JDK 8 ou supérieur
- Apache Ant (optionnel, pour la compilation)
